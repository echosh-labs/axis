[ axis mundi ]

A sophisticated open-source software repository designed to bridge the gap between Google Gemini's voice interface and Google Workspace.

By utilizing a high-performance Terminal User Interface (TUI), the system allows users to convert spoken thoughts into organized Google Keep notes and actionable tasks. The engine emphasizes intelligent automation, enabling the seamless coordination of complex workflows through both manual oversight and autonomous monitoring.

[ core features ]

Voice-to-Task Conversion

Transform spoken thoughts into organized Google Keep notes and actionable tasks using Google Gemini's voice interface.

Terminal User Interface

High-performance TUI for efficient workflow management and system orchestration.

Intelligent Automation

Seamless coordination of complex workflows with manual oversight and autonomous monitoring capabilities.

Real-time Synchronization

Server-Sent Events enable instantaneous data synchronization between backend and frontend.
